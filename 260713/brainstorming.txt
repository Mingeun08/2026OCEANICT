바다 데이터
국립해양조사원이나 해양환경정보포털(Open API)에서 수온·염분·파고 데이터를 가져와 자동 업데이트
어류 사진, 학명, 최적 수온, 먹이, 서식 깊이 등을 함께 표시

현재 수온·염분 조건(수온을 +2℃ 또는 염분을 변화시키면 어떤 어종의 서식 가능성이 달라지는지 시뮬레이션)을 입력하면 어떤 어종이 잘 서식할지 예측하는 간단한 머신러닝 모델(Random Forest 등)



해양 환경 변화가 어류 분포에 미치는 영향을 예측하는 AI 플랫폼

| 항목                                              | 의미                                   | 높을수록 의미하는 것                |
| ----------------------------------------------- | ------------------------------------ | -------------------------- |
| **COD (Chemical Oxygen Demand, 화학적 산소요구량)**     | 물속의 유기물과 산화 가능한 물질을 분해하는 데 필요한 산소의 양 | **높을수록 오염이 심함**            |
| **BOD (Biochemical Oxygen Demand, 생물학적 산소요구량)** | 미생물이 유기물을 분해하는 데 소비하는 산소의 양          | **높을수록 유기물 오염이 심함**        |
| **DO (Dissolved Oxygen, 용존산소)**                 | 물속에 녹아 있는 산소의 양                      | **높을수록 수질이 좋음**            |
| **TN (Total Nitrogen, 총질소)**                    | 물속에 존재하는 모든 형태의 질소(질산염, 암모니아 등)의 총량  | **너무 높으면 부영양화 발생**         |
| **TP (Total Phosphorus, 총인)**                   | 물속의 모든 형태의 인의 총량                     | **너무 높으면 녹조·적조 발생 가능성 증가** |
| **클로로필 a (Chlorophyll-a)**                      | 식물플랑크톤이 가진 엽록소의 양                    | **높을수록 플랑크톤이 많음**          |
| **TOC (Total Organic Carbon, 총유기탄소)**           | 물속 유기물에 포함된 탄소의 총량                   | **높을수록 유기물 오염이 많음**        |
